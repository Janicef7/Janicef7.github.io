Glosario Topologias de Red

Monotarea

Un sistema operativo que solo puede ejecutar un proceso a la vez.

Multitarea

Un sistema operativo que puede ejecutar varios procesos a la vez.

Multiusario

Un sistema operativo que permite que varios usuarios accedan a él al mismo tiempo.

Monousuario

Un sistema operativo que solo permite que un usuario acceda a él a la vez.

Real time

Un sistema operativo que debe responder a las solicitudes en un tiempo determinado.

No real time

Un sistema operativo que no tiene un tiempo de respuesta definido.

Sistema operativo de propósito general

Un sistema operativo que se puede utilizar para una variedad de tareas.

Sistema operativo de propósito específico

Un sistema operativo diseñado para una tarea específica.

Sistema operativo de tiempo compartido

Un sistema operativo que permite que varios usuarios accedan a un recurso compartido al mismo tiempo.

Sistema operativo multiprocesador

Un sistema operativo que puede ejecutar procesos en varios procesadores al mismo tiempo.

Sistema operativo distribuido

Un sistema operativo que gestiona varios ordenadores conectados en una red.

Sistema operativo de núcleo monolítico:

Un sistema operativo en el que todos los componentes están integrados en un único núcleo.

Sistema operativo de núcleo microkernel

Un sistema operativo en el que la mayoría de los componentes están aislados del núcleo.

Sistema operativo empotrado

Un sistema operativo diseñado para funcionar en dispositivos pequeños y con recursos limitados.

Sistema operativo de servidor

Una Un sistema operativo diseñado para servir a otros dispositivos en una red.

Sistema operativo de escritorio

Un sistema operativo diseñado para ser utilizado en ordenadores de sobremesa y portátiles.
